# 双指针<br>
# 题目<br>
给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。
示例 1：
输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]
# 思路<br>
- 暴力时时间复杂度是O（n+nlogn）
- 要解决的：虽然原数组按大小顺序排序，但平方后会改变顺序。怎么快速排结果序数组————直接从原数组入手，左右两头的绝对值都应该比较大，比较哪边更大插入目标数组就可以了<br>
- 也就是这里——双指针：原数组左右点各一指针，比较出平方后较大值插入目标数组后，大的指针向中间移动<br>
# 代码
```
int k = A.size() - 1;
        vector<int> result(A.size(), 0);
        for (int i = 0, j = A.size() - 1; i <= j;) { // 注意这里要i <= j，因为最后要处理两个元素
            if (A[i] * A[i] < A[j] * A[j])  {
                result[k--] = A[j] * A[j];
                j--;
            }
            else {
                result[k--] = A[i] * A[i];
                i++;
```
